# Air
공기처럼 가벼운 시간표 앱

본 프로젝트는 광고와 불필요한 기능으로 비대해진 시간표 앱의 사용에 불편함을 느끼는 유저들을 위해 만들었습니다.

# 기능
1. 1분, 5분, 60분단위 원하는대로 
2. 오늘의 요일을 라벨을 통해 바로 확인할 수 있습니다.
3. ~~친구의 시간표를 스크랩할 수 있습니다.

# 사용된 라이브러리
TimetableView:1.0.2

# 버그 픽스
1.0.1v - 9개 이상의 스케줄을 추가시 앱이 비정상 종료되는 문제를 수정했습니다.
