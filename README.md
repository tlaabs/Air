# Air
공기처럼 가벼운 시간표 앱

본 프로젝트는 너무 많은 광고와 불필요한 기능이 이것저것 붙어 비대해진 시간표 앱을 사용하는데 불편함을 느끼는 유저들을 위해 만들었습니다.

구글 플레이 다운로드

기능
1. 깔끔한 시간표 작성 기능
2. 요일에따른 테이블 레이블색 변경
3. 친구의 시간표를 보고 scrap할 수 있는 ㅣ능

사용한 라이브러리
'com.github.tlaabs:TimetableView:1.0.2'
https://github.com/tlaabs/TimetableView

버그 픽스
